# Ontario Vaccine Passport QR Code Parser

I wanted to see what was in the QR code for the Ontario COVID-19 vaccine passport.

## Dev Usage

```bash
pipenv install
pipenv shell
python main.py [image]
python main.py --textfile [qr_text]
```
